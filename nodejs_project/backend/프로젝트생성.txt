프로젝트 생성 절차
 > "npm install -g express-generator" 선행 후 가능.

1. 프로젝트를 만들 폴더로 이동한다.
	 >  "cd 경로"

2. 플젝명 폴더 생성.
	 > "express project명 --view=ejs"
	//위 명령어에서 --view 옵션은 Express에서 사용할 View Engine을 선택하는 옵션. 위 예제에서는 ejs를 사용하도록 설정.
	
3. 플젝명 폴더로 이동.
	 > "cd 플젝명"
	//새로 생긴 폴더로 이동. package.json 파일이 있는 폴더로 이동해야 한다. 아직 node_modules폴더 등 추가적인 모듈 없는 상태.

4. 의존성 모듈을 설치
	 > "npm install"
	 //-- package.json 파일 읽어서 node_modules폴더 만들고 필요한 추가 라이브러리가 설치됨.

5. 서버 시작.
	 > "npm start"
	or
	> "nodemon start" // nodemon 있는 경우. nodemon은 자동 설치 되는 프로그램이 아니라 없을 수도 있어요.

5. 게시판 만들기
- 확장자 ejs로 바꿔서 views 폴더에 놓는다
ex)
board_list.html
board_write.html
board_view.html
- routes 폴더에 라우터만들기.

6. 라우터 만들기 (board.js, member.js) : 업무별로 분할하여 라우터파일 생성
ex)
board.js
members.js
=>업무별로 라우터 파일 생성.

7. app.js 에서 라우터 import, require url과 등록

쿠키 - 내 컴퓨터에 저장되는 정보 -> 보안에 취약
세션 - 서버에 저장. (로그온이나 장바구니 등의 정보를 저장하기 위해서 사용하는 객체)

nodejs에서의 세션을 1.파일 2.디비
2번 mysql 디비에 세션정보를 저장하려고 한다

npm install express-session
npm install express-mysql-session

app.js 파일 열어서 입력
const session = require("express-session");
const MYSQLSTORE = require("express-mysql-session")(session);
